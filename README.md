# Bookmark Manager MCP Assignment

This project is a Bookmark Manager built with HTML, CSS, and JavaScript. It demonstrates a full Codex workflow using planning, issue generation, test-driven development, architecture review, and MCP integrations.

## Features
- Add bookmarks
- Edit bookmarks
- Delete bookmarks
- Search and filter bookmarks
- Local storage persistence
